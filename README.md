House Price Prediction

-> This project demonstrates a complete data analytics & machine learning pipeline using MySQL, Python, Power BI to analyze and predict house prices.

Goal:

Analyze real-estate data using SQL

Build a machine learning model to predict house prices

Visualize insights using Power BI & Tableau
